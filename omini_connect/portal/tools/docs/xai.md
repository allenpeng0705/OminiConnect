# xAI Tools

Provider: `xai` | Engine: `nango` | Auth: API Key via Nango

## Overview

xAI is an AI company building intelligent systems. **Requires api key via nango.**

## Authentication

**Nango API Key**:
- User provides their xAI API key
- Key stored securely in Nango

## Tool List

| Tool | Description | Method | Endpoint |
|------|-------------|--------|----------|
| `xai_get_model` | Get model information | GET | /v1/model |
| `xai_list_models` | List available models | GET | /v1/models |
| `xai_create_chat_completion` | Create a chat completion | POST | /v1/chat/completions |
| `xai_create_completion` | Create a text completion | POST | /v1/completions |
| `xai_create_embedding` | Create embeddings | POST | /v1/embeddings |
| `xai_get_usage` | Get API usage statistics | GET | /v1/usage |
| `xai_list_files` | List uploaded files | GET | /v1/files |
| `xai_upload_file` | Upload a file | POST | /v1/files |
| `xai_get_file` | Get file details | GET | /v1/files/{id} |
| `xai_delete_file` | Delete a file | DELETE | /v1/files/{id} |
