# Workday Adaptive Planning (Basic) Tools

Provider: `workday-adaptive-planning-basic` | Engine: `nango` | Auth: Basic Auth via Nango

## Overview

Workday Adaptive Planning with Basic Auth. **Requires basic auth via nango.**

## Authentication

**Nango Basic Auth**:
- User provides username/password
- Credentials stored securely in Nango

## Tool List

| Tool | Description | Method | Endpoint |
|------|-------------|--------|----------|
| `workday_list_plans` | List all planning plans | GET | /v1/plans |
| `workday_get_plan` | Get plan details | GET | /v1/plans/{id} |
| `workday_list_instances` | List all planning instances | GET | /v1/instances |
| `workday_get_instance` | Get instance details | GET | /v1/instances/{id} |
| `workday_list_sheets` | List all planning sheets | GET | /v1/sheets |
| `workday_get_sheet` | Get sheet details | GET | /v1/sheets/{id} |
| `workday_get_data` | Get planning data | GET | /v1/data |
| `workday_set_data` | Set planning data | POST | /v1/data |
| `workday_list_dimensions` | List all dimensions | GET | /v1/dimensions |
| `workday_list_levels` | List all levels in the hierarchy | GET | /v1/levels |
