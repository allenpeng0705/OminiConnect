# Triple Whale Tools

Provider: `triple-whale` | Engine: `nango` | Auth: API Key via Nango

## Overview

Triple Whale is an e-commerce analytics platform. **Requires api key via nango.**

## Authentication

**Nango API Key**:
- User provides their Triple Whale API key
- Key stored securely in Nango

## Tool List

| Tool | Description | Method | Endpoint |
|------|-------------|--------|----------|
| `triple_whale_list_stores` | List all connected stores | GET | /api/stores |
| `triple_whale_get_store` | Get store details | GET | /api/stores/{id} |
| `triple_whale_list_orders` | List all orders | GET | /api/orders |
| `triple_whale_get_order` | Get order details | GET | /api/orders/{id} |
| `triple_whale_list_products` | List all products | GET | /api/products |
| `triple_whale_get_product` | Get product details | GET | /api/products/{id} |
| `triple_whale_get_analytics` | Get analytics data | GET | /api/analytics |
| `triple_whale_list_customers` | List all customers | GET | /api/customers |
| `triple_whale_get_customer` | Get customer details | GET | /api/customers/{id} |
| `triple_whale_get_reports` | Get report data | GET | /api/reports |
