# Upsales Tools

Provider: `upsales` | Engine: `nango` | Auth: API Key via Nango

## Overview

Upsales is a sales CRM and marketing automation platform. **Requires api key via nango.**

## Authentication

**Nango API Key**:
- User provides their Upsales API key
- Key stored securely in Nango

## Tool List

| Tool | Description | Method | Endpoint |
|------|-------------|--------|----------|
| `upsales_list_companies` | List all companies | GET | /api/v2/companies |
| `upsales_get_company` | Get company details | GET | /api/v2/companies/{id} |
| `upsales_list_contacts` | List all contacts | GET | /api/v2/contacts |
| `upsales_get_contact` | Get contact details | GET | /api/v2/contacts/{id} |
| `upsales_list_deals` | List all deals/opportunities | GET | /api/v2/deals |
| `upsales_get_deal` | Get deal details | GET | /api/v2/deals/{id} |
| `upsales_create_deal` | Create a new deal | POST | /api/v2/deals |
| `upsales_list_activities` | List all activities | GET | /api/v2/activities |
| `upsales_create_activity` | Create a new activity | POST | /api/v2/activities |
| `upsales_get_analytics` | Get sales analytics | GET | /api/v2/analytics |
