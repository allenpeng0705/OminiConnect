# Unauthenticated Tools

## Overview

No authentication required for this integration. **Requires no authentication.**

## Authentication

## Tool List

| Tool | Description | Method | Endpoint |
|------|-------------|--------|----------|
| `unauthenticated_health_check` | Check service health | GET | /health |
| `unauthenticated_get_info` | Get service information | GET | /info |
| `unauthenticated_list_public` | List public data | GET | /public |
| `unauthenticated_get_public` | Get public data by ID | GET | /public/{id} |
| `unauthenticated_search` | Search public data | GET | /search |
| `unauthenticated_list_resources` | List available resources | GET | /resources |
| `unauthenticated_get_resource` | Get a specific resource | GET | /resources/{id} |
| `unauthenticated_list_tags` | List available tags | GET | /tags |
| `unauthenticated_get_status` | Get service status | GET | /status |
| `unauthenticated_list_versions` | List API versions | GET | /versions |
