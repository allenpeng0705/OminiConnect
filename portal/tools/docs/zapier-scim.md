# Zapier SCIM Tools

Provider: `zapier-scim` | Engine: `nango` | Auth: API Key via Nango

## Overview

Zapier SCIM API for user provisioning. **Requires api key via nango.**

## Authentication

**Nango API Key**:
- User provides their Zapier SCIM API key
- Key stored securely in Nango

## Tool List

| Tool | Description | Method | Endpoint |
|------|-------------|--------|----------|
| `zapier_list_users` | List all users | GET | /Users |
| `zapier_get_user` | Get user details | GET | /Users/{id} |
| `zapier_create_user` | Create a new user | POST | /Users |
| `zapier_update_user` | Update a user | PUT | /Users/{id} |
| `zapier_delete_user` | Delete a user | DELETE | /Users/{id} |
| `zapier_list_groups` | List all groups | GET | /Groups |
| `zapier_get_group` | Get group details | GET | /Groups/{id} |
| `zapier_create_group` | Create a new group | POST | /Groups |
| `zapier_update_group` | Update a group | PUT | /Groups/{id} |
| `zapier_delete_group` | Delete a group | DELETE | /Groups/{id} |
