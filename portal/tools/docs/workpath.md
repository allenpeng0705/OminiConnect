# Workpath Tools

Provider: `workpath` | Engine: `nango` | Auth: API Key via Nango

## Overview

Workpath is an HR software platform for performance management. **Requires api key via nango.**

## Authentication

**Nango API Key**:
- User provides their Workpath API key
- Key stored securely in Nango

## Tool List

| Tool | Description | Method | Endpoint |
|------|-------------|--------|----------|
| `workpath_list_employees` | List all employees | GET | /employees |
| `workpath_get_employee` | Get employee details | GET | /employees/{id} |
| `workpath_list_goals` | List all goals | GET | /goals |
| `workpath_get_goal` | Get goal details | GET | /goals/{id} |
| `workpath_create_goal` | Create a new goal | POST | /goals |
| `workpath_list_reviews` | List all performance reviews | GET | /reviews |
| `workpath_get_review` | Get review details | GET | /reviews/{id} |
| `workpath_list_teams` | List all teams | GET | /teams |
| `workpath_get_team` | Get team details | GET | /teams/{id} |
| `workpath_list_activities` | List all activities | GET | /activities |
